¡Bienvenido a mi primer post en Astro!

Astro es un framework moderno que está revolucionando el desarrollo web gracias a su enfoque en la velocidad y la optimización. Permite crear sitios rápidos, eficientes y fáciles de mantener, combinando lo mejor de tecnologías como JavaScript, TypeScript y componentes de diferentes frameworks (React, Vue, Svelte, etc.).

### ¿Por qué elegir Astro?

- **Rendimiento superior:** Astro genera páginas estáticas por defecto, lo que reduce el tiempo de carga y mejora la experiencia del usuario.
- **Flexibilidad:** Puedes usar componentes de distintos frameworks en un mismo proyecto.
- **Simplicidad:** Su sintaxis es clara y fácil de aprender, ideal tanto para principiantes como para desarrolladores experimentados.

### Características principales

- Generación de contenido estático y dinámico.
- Integración sencilla con Markdown y archivos de contenido.
- Soporte para TypeScript y herramientas modernas de desarrollo.
- Sistema de rutas intuitivo y personalizable.
